## 概述

本版本为 PROJECT-001（热文采集改写平台）的 P0 功能交付版本，核心新增**多平台一键发布**功能。用户可在文章改写完成后，一键发布到微信公众号和知乎专栏。

## 核心功能

### ✨ 新增功能

| 功能 | 描述 |
|------|------|
| **多平台发布 API** | `POST /api/publish` — 创建发布任务；`GET /api/publish/{task_id}` — 查询发布进度 |
| **微信公众号发布器** | `WechatPublisher` — 获取 access_token、上传封面、创建草稿、正式发布 |
| **知乎专栏发布器** | `ZhihuPublisher` — 获取 access_token、发布文章到知乎专栏 |
| **BasePublisher 基类** | 统一错误处理+重试逻辑，降低各平台发布器实现成本 |
| **前端一键发布 UI** | 改写完成后显示平台选择 + 发布按钮 + 实时进度显示 |

## 技术架构

```
改写完成 → 显示一键发布区域
    ↓
选择平台（微信/知乎）→ 点击发布
    ↓
POST /api/publish { article_id, platforms }
    ↓
创建发布任务 (task_id) → 返回 task_id
    ↓
轮询 GET /api/publish/{task_id}
    ↓
每个平台独立执行发布器
    ├── WechatPublisher: get_access_token → upload_media → create_draft → publish
    └── ZhihuPublisher:  get_access_token → publish_article
```
